Here's a comprehensive list of HTML tags with examples and all possible attributes, one per line:

1. **`<!DOCTYPE>`** - Defines document type
   ```html
   <!DOCTYPE html>
   ```

2. **`<a>`** - Anchor/hyperlink
   ```html
   <a href="https://example.com" target="_blank" rel="noopener" download="file.pdf" hreflang="en" type="text/html" referrerpolicy="no-referrer" ping="https://tracker.com" media="screen" charset="UTF-8" coords="x,y" shape="rect" accesskey="k" tabindex="1" title="Tooltip" id="link1" class="nav-link" style="color: blue;" data-custom="value" aria-label="Description">Click me</a>
   ```

3. **`<abbr>`** - Abbreviation
   ```html
   <abbr title="World Health Organization" id="who" class="abbrev" style="border-bottom: 1px dotted;" data-info="health">WHO</abbr>
   ```

4. **`<address>`** - Contact information
   ```html
   <address id="contact" class="contact-info" style="font-style: normal;" data-type="business">123 Main St</address>
   ```

5. **`<area>`** - Image map area
   ```html
   <area shape="rect" coords="34,44,270,350" href="page.html" alt="Description" target="_blank" rel="noopener" download="file" hreflang="en" type="text/html" media="screen" ping="https://tracker.com" accesskey="a" tabindex="2" title="Area title" id="area1" class="image-area" data-custom="value">
   ```

6. **`<article>`** - Self-contained content
   ```html
   <article id="article1" class="blog-post" style="margin: 10px;" lang="en" dir="ltr" data-category="tech" itemscope itemtype="https://schema.org/Article">Content</article>
   ```

7. **`<aside>`** - Side content
   ```html
   <aside id="sidebar" class="sidebar" style="float: right;" lang="en" dir="ltr" data-position="right" hidden>Side content</aside>
   ```

8. **`<audio>`** - Audio player
   ```html
   <audio src="audio.mp3" controls autoplay loop muted preload="auto" crossorigin="anonymous" id="audio1" class="audio-player" style="width: 100%;" data-duration="120">Your browser doesn't support audio</audio>
   ```

9. **`<b>`** - Bold text
   ```html
   <b id="bold1" class="bold-text" style="font-weight: bold;" lang="en" dir="ltr" title="Bold text" data-importance="high">Bold text</b>
   ```

10. **`<base>`** - Base URL
    ```html
    <base href="https://example.com/" target="_blank">
    ```

11. **`<bdi>`** - Bi-directional isolation
    ```html
    <bdi id="bdi1" class="bdi-element" style="direction: rtl;" lang="ar" dir="auto" data-language="arabic">Text</bdi>
    ```

12. **`<bdo>`** - Bi-directional override
    ```html
    <bdo id="bdo1" class="bdo-element" style="direction: rtl;" lang="he" dir="rtl" data-direction="override">Text</bdo>
    ```

13. **`<blockquote>`** - Block quotation
    ```html
    <blockquote id="quote1" class="quotation" style="margin: 20px;" cite="https://source.com" lang="en" dir="ltr" data-author="Author Name">Quoted text</blockquote>
    ```

14. **`<body>`** - Document body
    ```html
    <body id="body" class="main-body" style="margin: 0;" lang="en" dir="ltr" onload="init()" onunload="cleanup()" onscroll="handleScroll()" onresize="handleResize()" data-theme="dark">Content</body>
    ```

15. **`<br>`** - Line break
    ```html

    ```

16. **`<button>`** - Clickable button
    ```html

    ```

17. **`<canvas>`** - Graphics container

    ```html
    <var id="var1" class="variable" style="font-style: italic;" lang="en" dir="ltr" data-type="mathematical">x</var>
    ```

105. **`<video>`** - Video player
    ```html
    <video id="video1" class="video-player" src="video.mp4" width="640" height="360" style="max-width: 100%;" poster="poster.jpg" preload="auto" autoplay loop muted controls playsinline crossorigin="anonymous" mediagroup="group1" onplay="handlePlay()" onpause="handlePause()" data-duration="120">Your browser doesn't support video</video>
    ```

106. **`<wbr>`** - Line break opportunity
    ```html
    <wbr id="wbr1" class="word-break" style="display: inline;" lang="en" dir="ltr" data-type="break">
    ```
